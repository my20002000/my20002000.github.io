安装Hexo和主题

发表于 2021-01-07

安装

mkdir www
cd www
yarn global add hexo-cli
yarn add hexo-deployer-git
hexo init blog
cd blog
git clone https://github.com.cnpmjs.org/theme-next/hexo-theme-next themes/next --depth=1
#修改blog/_config.yml>language: zh-CN (第11行)
#修改blog/_config.yml>url: https://[username].github.io/ (第16行)
#修改blog/_config.yml>theme: next (第101行)
#修改blog/_config.yml(最后)
deploy:
  type: 'git'
  repo: git@github.com:[username]/[username].github.io.git
  branch: master
#修改blog/themes/next/_config.yml>display: always(第154行)(sidebar:节点下)
echo "Welcome to [https://[username].github.io](https://[username].github.io)! ">./source/README.md
#修改blog/_config.yml>skip_render: README.md(第32行)

git配置

C:
cd %userprofile%
mkdir .ssh
cd .ssh
git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub注册邮箱"
ssh-keygen -t rsa -C "你的GitHub注册邮箱"
#复制id_rsa.pub 到 GitHub_Settings_keys
